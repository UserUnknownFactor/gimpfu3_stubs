Description [src]

already called from gimp_procedure_config_end_export().

Only use this function if the GimpMetadata returned by gimp_procedure_config_begin_run() needs to be written at a specific point of the export, other than its end.

This function syncs back config‘s export properties to the metadata’s GimpMetadataSaveFlags and writes the metadata to file using gimp_image_metadata_save_finish().

The metadata is only ever written once. If this function has been called explicitly, it will do nothing when called a second time from gimp_procedure_config_end_export().
